A string is hidden inside the git history of a public repository. The string is no longer in any current file. It was committed, then later overwritten and the file was deleted.
The repository:
github.com/thebigjah/prowess-archive
Recover the string. It looks like WORD-NNNN-WORD.
You will want some combination of:
· git clone
· git log --all --oneline
· git log -p -- <path> (shows all history of a path including deletions)
· git log -S <pattern> --all (pickaxe: find commits that added or removed a string)
· git show <sha>
The next path is /prowess/<p>/ where p is the first 16 hex of sha256(string + 'finish'). Pass the string exactly as it appears in history — same case, same hyphens.
